CrochetCraft is a responsive e-commerce website created for showcasing and selling handmade crochet products.
This project was built as part of my UI/UX and Web Development internship to practice real-world website design using HTML, CSS, and Bootstrap 5.
The website includes a homepage, product section, and contact page with a clean and aesthetic user interface.

🛠️ Technologies Used
HTML5
CSS3
Bootstrap 5
Responsive Web Design
Flexbox & Grid Layout

✨ Features
Fully responsive design (mobile, tablet, desktop)
Modern and clean UI design
Product cards with images and price
Navigation bar with multiple pages
Handmade crochet product showcase
Contact page for user interaction
